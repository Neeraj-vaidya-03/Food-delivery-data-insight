# Food-delivery-data-insight
Interactive Power BI dashboard for food delivery operations. Tracks on-time performance, regional customer satisfaction, and app-specific delivery trends using custom DAX measures and interactive visualizations.
Technical Highlights:

Data Modeling: Established relationships between delivery data and calendar dimensions to enable time-based filtering.

DAX Measures: Implemented custom measures for dynamic period reporting and performance status evaluation.

Visualizations:

Developed custom line charts with smoothed interpolation to visualize operational trends.

Designed high-contrast KPI cards with conditional formatting for instant performance tracking (on-time vs. late metrics).

Utilized interactive slicers to allow users to filter data by region and specific delivery applications.

Key Metrics Analyzed:

Total Order Volume

On-Time Delivery %

Late Delivery Counts

Average Customer Rating by City
